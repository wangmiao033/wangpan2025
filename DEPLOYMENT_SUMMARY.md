## 🌐 部署选项

### 推荐部署方式

1. **Heroku** (最简单)
   ```bash
   heroku create your-app-name
   git push heroku main
   ```

2. **Vercel** (静态托管)
   ```bash
   vercel --prod
   ```

3. **Netlify** (拖拽部署)
   - 访问 netlify.com
   - 拖拽项目文件夹

4. **Railway** (自动部署)
   - 连接 GitHub 仓库
   - 自动检测并部署

### 本地测试
```bash
npm install
npm start
# 访问 http://localhost:3000
```

## 🆘 常见问题解决

### 部署问题
- **端口冲突**: 修改 `PORT` 环境变量
- **依赖安装失败**: 检查 Node.js 版本
- **文件上传失败**: 检查文件大小限制

- **拖拽不工作**: 检查浏览器兼容性
